Aircraft Tracker

Overview: Developed for the Circuit Communications class, this project utilizes a USB RTL-SDR dongle connected to an antenna to capture live radio frequency signals from nearby aircraft.

How it Works: The system intercepts Automatic Dependent Surveillance–Broadcast (ADS-B) signals. By processing these raw signals, the system extracts critical data such as the aircraft's type and unique callsign.

Technical Details: The system relies on a feedback loop where motors adjust their speed and direction based on real-time data.

FPGA Reaction Timer

Overview: A Microprocessor Architecture project built on an Intel FPGA using Verilog, simulating the "start lights" found in racing.

Functionality: The board's LEDs simulate a countdown sequence. Once the light turns green, an internal millisecond timer begins. The user stops the timer via a physical button, and the precise reaction time is displayed on the board's 7-segment display modules.
