Aircraft Tracker

Overview: Developed for the Circuit Communications class, this project utilizes a USB RTL-SDR dongle connected to an antenna to capture live radio frequency signals from nearby aircraft.

Technical Description: The system intercepts Automatic Dependent Surveillance–Broadcast (ADS-B) signals at 1090 MHz. By processing these raw IQ signals, the system extracts critical data frames containing the aircraft's ICAO address, callsign, altitude, and velocity. This project focused purely on hardware configuration and signal processing theory.

Self Balancing Robot

Overview: Created for the Embedded Systems class, this robot repurposes a 3-wheel chassis into a 2-wheel self-balancing vehicle.

Technical Description: Controlled by a Texas Instruments TIVA LaunchPad, the system implements a PID controller loop. It reads orientation data from an IMU (Inertial Measurement Unit) via I2C and calculates the necessary motor torque to maintain specific angles. PWM signals drive the DC motors to correct tilt in real-time.

Water Monitoring System

Overview: A Sensor Applications class project designed to read and visualize water quality data remotely.

Technical Description: Analog sensors measure turbidity and temperature. An Arduino microcontroller performs ADC conversion and transmits raw data strings over UART to a host PC. A custom LabVIEW application parses this data, displays it on a local dashboard, and makes HTTP POST requests to the ThingSpeak API for cloud logging.

FPGA Reaction Timer

Overview: A Microprocessor Architecture project built on an Intel FPGA using Verilog, simulating the "start lights" found in racing.

Technical Description: Implemented using Verilog HDL. The logic includes a Linear Feedback Shift Register (LFSR) for random wait times, a millisecond counter driven by the board's 50MHz clock, and a Binary-to-BCD converter to drive the 7-segment displays. State machines manage the IDLE, COUNTDOWN, and TIMING states.

Smart Parking System

Overview: Developed for the Digital Systems class, this system automates the tracking of parking availability using an Arduino and infrared sensors.

Technical Description: IR obstruction sensors are installed at each bay and the entrance/exit. The Arduino manages an interrupt-based counter to track total occupancy. It updates a 16x2 I2C LCD to display free slots dynamically, preventing drivers from entering a full lot.

Line Following Robot

Overview: Developed for Circuits 2, this autonomous robot navigates a track by detecting contrast between the floor and a line.

Technical Description: Uses an L298N motor driver and an Arduino. Two IR proximity sensors (comparators) detect the black line against a white surface. A bang-bang control algorithm adjusts the differential steering: if the left sensor sees black, the robot steers left to center itself.

Laser Security System

Overview: My first university project (Circuits 1), exploring analog circuit design for security applications.

Technical Description: A purely analog circuit using a Light Dependent Resistor (LDR) in a voltage divider configuration. The voltage from the divider feeds into the base of a BJT transistor. When the laser beam is broken, the LDR resistance rises, increasing the base voltage and switching the transistor ON to drive a 5V buzzer.

Tiny-VGG Classifier

Overview: A lightweight Computer Vision model implementation using PyTorch to classify images with high accuracy.

Technical Description: This project replicates the famous VGG (Visual Geometry Group) architecture, scaled down to a "Tiny" version for efficient training. Built with PyTorch, it features sequential convolutional blocks (Conv2d, ReLU, MaxPool) followed by a classifier head. The project includes a custom dataset loader, training loop visualization (Loss/Accuracy), and model evaluation scripts.
